Project Overview:
•	This project analyses long-term agricultural productivity trends across Indian states (1997–2020) using SQL and Power BI. 
•	The objective is to identify regional yield growth patterns, seasonal performance differences, and structural productivity shifts to support data-driven agricultural decision-making.
Problem Statement:
1.	Which states have improved or declined in yield over decades?
2.	How does seasonal cropping impact productivity?
3.	Does rainfall strongly influence yield performance?
4.	Are high-production states also high-efficiency states?
Tools & Technologies:
•	MySQL – Data cleaning, aggregation, and window functions
•	Power BI – Data modelling, DAX measures, and dashboard development
•	DAX – Time-based yield comparison (2010s vs 1990s), KPI creation
•	Statistical Analysis – Volatility (STDEV), weighted yield metrics
Key Analytical Metrics:
•	Weighted Yield (Production / Area)
•	Yield Change (2010s vs 1990s)
•	Seasonal Average Yield
•	Yield Volatility (Standard Deviation)
•	State-wise Production Distribution
Key Insights:
1.	Yield growth varies significantly across states, indicating regional efficiency disparities.
2.	Telangana demonstrates strong long-term yield improvement.
3.	Maharashtra shows noticeable yield contraction over decades.
4.	Winter cropping exhibits higher and more stable yields compared to Kharif.
5.	Rainfall alone does not strongly predict agricultural productivity.
Business Recommendations:
1.	Conduct targeted agronomic efficiency audits in declining states.
2.	Promote winter-season best practices to improve yield stability.
3.	Optimize fertilizer allocation strategies in high-volatility regions.
